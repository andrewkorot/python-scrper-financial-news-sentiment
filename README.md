# Financial News Sentiment Analyzer

A Python project that scrapes real-time financial news headlines and analyzes market sentiment using NLP.

## What it does
- Scrapes 300+ headlines for AAPL, TSLA, and GOOGL from Finviz
- Cleans and structures the data using Pandas
- Runs sentiment analysis using TextBlob (NLP)
- Visualizes daily sentiment trends with Matplotlib
- Exports a full dataset (CSV) and automated report (TXT)

## Tools Used
- Python, BeautifulSoup, Pandas, TextBlob, Matplotlib
- Google Colab

## Key Findings (Apr 21-28, 2026)
- 300 headlines analyzed across 3 major tech stocks
- TSLA had the most negative coverage (16 negative headlines)
- AAPL had the most positive coverage (29 positive headlines)
- 64% Neutral, 25.3% Positive, 10.7% Negative

## How to Run
1. Open the notebook in Google Colab
2. Run all cells in order
3. Outputs: CSV dataset, 3 charts, summary report
